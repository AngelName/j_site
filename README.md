这个项目是一个通过create react app 创建后的项目的拓展手脚架

环境系统: macOS 10.14.4
node版本: node v10.15.0

## 目的
为了构建一个轻量的功能比较全的，可以开箱即用的react 手脚架

### `npm start`

启动项目

### `npm start`

测试项目

### `npm run build`

构建到生残环境

### `npm run eject`

把配置文件暴露出来，默认的配置就不生效了，全由自己配置
（webpack babel eslint）
